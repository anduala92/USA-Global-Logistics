# Презентация на проекта: USA Global Logistics

## 1) Проблем и решение

**Проблем:**
Малки и средни логистични компании често управляват процесите и ресурсите си в различни инструменти, което води до грешки и лоша проследяемост.

**Решение:**
Единна уеб система за управление на клиенти, поръчки, превозвачи, шофьори, МПС и пратки, с ролеви достъп и интеграция чрез API.

## 3) Таблица на API endpoint-и

| Endpoint               | Метод | Описание                            |
|------------------------|-------|-------------------------------------|
| /auth/register         | POST  | Регистрация (email, password, role) |
| /auth/login            | POST  | Вход (email, password, deviceInfo)  |
| /customers             | CRUD  | Работа с клиенти                    |
| /orders                | CRUD  | Работа с поръчки                    |
| /vehicles              | CRUD  | Работа с МПС                        |
| /carriers              | CRUD  | Работа с превозвачи                 |
| /drivers               | CRUD  | Работа с шофьори                    |
| /shipments             | CRUD  | Работа с пратки                     |

Пълният списък — виж `API_Endpoints.md`.
